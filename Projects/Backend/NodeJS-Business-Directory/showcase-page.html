NodeJS: Business Directory

Project Technologies:

Business Directory - Backend Project.

This project exemplifies the application of NodeJS, Express & MongoDB into a functional Backend for a Business Directory, Showcasing a RESTful API for managing businesses and users, user authentication and authorization, role-based access control, and data seeding for initial setup. It has been developed using NodeJS, MongoDB, Express, JWT, Joi, bcryptjs, Mongoose, Morgan, Postman, etc.
